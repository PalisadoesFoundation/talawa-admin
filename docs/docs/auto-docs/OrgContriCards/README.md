# OrgContriCards

A React functional component that displays contribution details for an organization.

## Component

## Param

The properties passed to the component.

## Param

The name of the user who made the contribution.

## Param

The email address of the user who made the contribution.

## Param

The date when the contribution was made.

## Param

The transaction ID associated with the contribution.

## Param

The amount contributed by the user.

## Remarks

- This component uses the `react-i18next` library for internationalization.
- The `styles` object is imported from a CSS module for styling.
- The component is designed to be used within the Talawa Admin project.

## Example

```tsx
<OrgContriCards
  userName="John Doe"
  userEmail="john.doe@example.com"
  contriDate="2023-01-01"
  contriTransactionId="12345ABC"
  contriAmount={100}
/>
```
