IUseDataTableFilteringOptions

Admin Docs

Interface: IUseDataTableFilteringOptions<T>

Defined in: src/types/shared-components/DataTable/hooks.ts:96

Configuration options for table data filtering and search functionality.

Provides comprehensive filtering capabilities including global search across all rows, per-column filtering, and control over client-side vs server-side filtering behavior. Supports pagination mode detection to manage page reset behavior during filtering.
