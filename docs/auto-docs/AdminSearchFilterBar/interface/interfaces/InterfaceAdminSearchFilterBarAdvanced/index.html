Interface: InterfaceAdminSearchFilterBarAdvanced

Defined in: src/types/AdminSearchFilterBar/interface.ts:294

Configuration for AdminSearchFilterBar with search and dropdown functionality.

Use this variant when you need search capabilities combined with one or more sorting/filtering dropdowns.
